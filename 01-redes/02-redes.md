# Redes - Aula 02: Endereços Ip`s 

## O que é um endereço IP?

IP(Internet Protocol), É um número usado para identificar o dispositivo.

## Objetivo:

Identificar os dispositivos conectados dentro de uma rede.

## Conceitos Importantes:

- possui dois tipos IP privado e IP público.
- IP público (É usado na comunicação de internet).
- IP privado (É usado dentro de uma rede local).
- IP estático permanece o mesmo.
- IP dinâmico pode mudar automaticamente.
- IPv4 É o formato tradicional,mais comum.
- IPv6 Foi criado para suportar mais dispositivos.

## Explicação:

Você entra em um site, seu computador envia uma solicitação usando seu endereço de ip pro servidor. o servidor responde enviando para o mesmo 
endereço de ip. permitindo que a pagina seja carregada.

## funcionamento:

- O dispositivo recebe um endereço IP.
- Ele envia uma solicitação pela rede.
- Os roteadores analisam o endereço de destino.
- Os dados são encaminhados até o destino correto.
- O servidor envia a resposta para o endereço IP de origem.

## Exemplo pratico:

Seu computador, ele utiliza seu endereço de ip para se comunicar com o servidor do site. O servidor recebe a solicitação e envia
a resposta para o mesmo endereço de ip, permitindo que a página seja exibida.

## resumo:

Bom o IP é isso, identifica dispositivos na rede. permiti a comunicação entre dispositivos. fora isso existe dois padrões principais IPv4 tradicional e o IPv6 mais moderno
Os roteadores utilizam os endereços de IP para encaminhar os dados.
