# O que é?

Se o IP indentifica quem é o dispositivo dentro de uma rede, o MAC diz quem ele é! MAC é a identidade da interfadce de rede.
O endereço MAC nada mais é do que a identidade do seu dipositivo na rede. O IP pode mudar, mas o MAC permanece o mesmo. 

# objetivo:

- Ele identifica o dispositivo na rede LAN. 
- Permitir a comunicação entre equipamentos conectados à mesma rede.
- Evitar que dois dispositivos sejam confundidos na rede.
  

# Conceitos importantes:

- Cada placa de rede possui um endereço MAC único.
- O endereço MAC é utilizado na Camada 2 (enlace) do modelo OSI.
- Diferente do endereço IP, o MAC normalmente não muda.
- Roteadores utilizam IP para comunicação entre redes, enquanto switches utilizm MAC para encaminhar dados na rede local.

# Explicação:

Quando um computador envia dados para outro na mesma rede. O switch verifica o endereço MAC de destino para encaminhar
os dados ao dispositivo correto. O endereço MAC funciona como uma "identidade" da placa de rede.

# Funcionamento:

- Um dispositivo envia dados para outro.
- Os switch varifica o endereço MAC de destino.
- O switch encaminha os dados apenas para a porta onde o dispositivo está conectado.
- O dispositivo recebe os dados.

# Exemplo Prático:

Imagine uma empresa com varios computadores ligados ao mesmo switch.
Cada computador possui um endereço MAC diferente.
Quando um computador envia um arquivo para outro, o switch identifica o endereço MAC de destino e entrega os dados somente ao computador correto.

# Resumo:

- O endereço MAC identifica uma placa de rede.
- É único para cada dispositivo.
- Atua na camada 2 do Modelo OSI
- É utilizado principalmente em redes locais.
- Os switches utilizam endereços MAC para encaminhar os dados.

# O que preciso lembrar?

- MAC significa Media Acess Control.
- É um identificador Único da placa de rede.
- Funciona na Camada 2 do Modulo.
- É usado em redes locais (LAN).
- Swichtes trabalham com endereços MAC.

# Palavras-Chaves:

- Endereço MAC.
- Media Access Control.
- Placa de rede.
- Switch.
- Camada 2
- Modelo OSI
- LAN.

# DICA PARA MEMORIZAR:

- MAC = "Quem é o dispositivo?" > Identifica a placa de rede na rede local
- IP = "Onde está o dispositivo?" > Identifica a localização do dispositivo na rede.
